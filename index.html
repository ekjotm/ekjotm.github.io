My Synth

This is a virtual sine wave synthesizer app I created for iOS in Swift. It uses UIKit for the UI and the sound elements came from the open source AudioKit (link to that below). The synth is playable on one screen for two octaves, and I plan on adding buttons to shift octaves up and down as you need. Making this taught me all the mentioned frameworks, . I want to expand on this app now by adding a graphic midi recording and editing feature so that you can create and export melodies on the go wherever you are and use them later in a proper full scale DAW on a computer!
